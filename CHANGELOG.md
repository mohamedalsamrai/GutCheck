# Changelog

All notable changes to GutCheck will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-03-22

### Added
- Initial release of GutCheck
- Cross-platform support: iOS, Android, Web (PWA), Linux, macOS
- Local-first database with Isar (native) and Hive (web)
- Meal logging with photo capture
- Wellness score tracking
- Detailed analytics and trend charts
- Data export functionality
- Progressive Web App (PWA) support
- Ingredient database with quick search
- Localization support (i18n)
- Offline-first functionality

### Known Issues
- Isar web support limited due to JavaScript 64-bit integer constraints
- iOS PWA requires cache clearing for icon updates

## [Unreleased]
